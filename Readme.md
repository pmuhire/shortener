
# Shortener BE
An express prisma node.js Typescript Postgress url Shortener
## Installation

Install my-project with npm

```bash
  git clone https://github.com/pmuhire/shortener.git
  cd shortener
  npm install
  npx prisma 
```
### Build
```bash
  npm run build
```
### Start
```
  npm start
```
### Dev mode
 ```
  npm run dev 
  ```
